# Yuta Portfolio

This project is a personal portfolio website for Yudhistira Hutama Putra. It includes various sections that showcase personal information, skills, achievements, and favorite characters (waifus). The website is designed to be visually appealing and user-friendly, with a login page that secures access to the main content.

## Project Structure

The project consists of the following files and directories:

- `index.html`: The login page where users enter a PIN to access the portfolio.
- `home.html`: The main landing page that introduces the user and provides personal information and hobbies.
- `about.html`: A page detailing the user's achievements in an accordion format for easy navigation.
- `skills.html`: A showcase of the user's skills, presented in a card format with images and descriptions.
- `waifu.html`: A list of the user's favorite characters (waifus) with images and descriptions, also in card format.
- `css/styles.css`: Custom styles for enhancing the visual appearance of the HTML pages.
- `js/main.js`: JavaScript code for handling login functionality, including PIN validation and session management.
- `gambar/`: A directory containing image files used throughout the project, such as background images and character images.
- `.gitignore`: Specifies files and directories to be ignored by Git version control.
- `README.md`: Documentation for the project, including setup instructions and an overview.
- `netlify.toml`: Configuration settings for deploying the project on Netlify, including build settings and redirects.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser to access the login page.
4. Enter the correct PIN to access the main content of the portfolio.

## Usage

- Navigate through the portfolio using the navigation links provided in the navbar.
- Each section is designed to be responsive and user-friendly.
- The login functionality ensures that only authorized users can access the main content.

## Deployment

This project can be deployed on platforms like Netlify. The `netlify.toml` file contains the necessary configuration for deployment.
